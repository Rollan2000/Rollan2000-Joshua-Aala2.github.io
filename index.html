LightBox

Intro

The Lightbox library displays video and images by filling the screen and dimming out the rest of the webpage.

NSB Media LightBox plugins are a helpful designer & developer tool. It is a way to add media to any website. It is mobile friendly and a way to integrate single images, galleries, slide shows,videos, iframes & custom content.

Features

Duration The time it takes for the Lightbox container and overlay to fade in and out, in milliseconds.
Image Fade Duration The time it takes for the image to fade in once loaded, in milliseconds.
Album Label The text displayed below the caption when viewing an image set. The default text shows the current image number and the total number of images in the set.
Image Slider An Image Slider that comprises several images displayed on a web application is called “Slide Show” and “Carousels”. It permits you to show multiple images, videos, or graphics on a web application. Also, adding an Image Slider to a website can fascinate the users and capture their attention.
Disable Scrolling If true, when the Lightbox is open, prevent the page from scrolling. The overflow is hidden on the body so the settings don't overflow.
Wrap Around If true,When a user reaches the last image in a set, the right navigation arrow will appear and they will be to continue moving forward which will take them back to the first image in the set.
Show Image Number Label If false, the text indicating the current image number and the total number of images in set (Ex. "image 2 of 4") will be hidden.

Tutorial

  1. Build HTML & CSS
  2. Download Light Box File
  3. In src folder link lightbox.css file in head tag
  4. Add JavaScript before closing body tag
  5. Load Jquery (compatible with Jquery 1.7 and up)
  6. If you dont have jquery load lightbox-plus-jquery.js file
  7. Make sure image files are linked correctly
  8. Add data-lightbox to enable lightbox (if it is a set attribute needs to be the same)
